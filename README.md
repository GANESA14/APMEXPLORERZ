# APM Explorerz – React App

## Quick Start
```
npm install
npm run dev
```

## Production Build
```
npm run build
npm run preview
```

## Media Files
Place your original media from the APM project into /public:
- `public/videoplayback.mp4` — hero background video
- `public/bikes/` — bike images
- `public/Album/` — album photos & videos
- `public/Riders/` — rider photos
- `public/Scroll/` — scroll section images

## Pages
| Route | Page |
|-------|------|
| / | Home |
| /bikes | Bikes |
| /album | Album |
| /videos | Videos |
| /trip | Trip & Routes |
| /riders | Riders |
| /about | About |
| /developer | Developer |

## Features
- ✅ Zero load screen (no preloader)
- ✅ React Router SPA
- ✅ Dark/Light theme (persisted)
- ✅ Draggable circular floating nav
- ✅ GSAP animations + ScrollTrigger
- ✅ AOS scroll animations
- ✅ Interactive Leaflet maps (Trip page)
- ✅ Three.js crystal (Developer page)
- ✅ Swiper carousel (About page)
- ✅ Album lightbox with keyboard nav
- ✅ Video filters with GSAP transitions
